Evergreen MapLibre Package v5

What this version does:
- Opens centered on the full set of test units
- Shows all test units by default
- Clicking a unit shows only nonzero artifact categories
- Popup categories are sorted from highest to lowest count
- Optional filter highlights units by a selected category

How to test locally:
1. Unzip this folder.
2. Open a terminal in the folder.
3. Run:
   python -m http.server 8000
4. Open:
   http://localhost:8000/

Deployment:
Upload all files in this folder to the same directory on your website.
